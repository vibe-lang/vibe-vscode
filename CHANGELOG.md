# Changelog

## 0.1.0

- Syntax highlighting for all Vibe language constructs
- String interpolation highlighting (`${expression}`)
- Code snippets for functions, classes, structs, enums, control flow, and more
- Auto-indentation for block keywords (`def`, `if`, `class`, `struct`, etc.)
- Bracket matching and auto-closing
- Code folding for `def`/`end`, `class`/`end`, `if`/`end`, etc.
- Comment toggling with `#`
- IntelliSense completions for 60+ built-in functions, keywords, and types
- Hover documentation for all built-in functions
